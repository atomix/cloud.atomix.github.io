    LogService

    LogService log service

    Method Name Request Type Response Type Description
    Create CreateRequest CreateResponse Create creates a log
    Close CloseRequest CloseResponse Close closes a log
    Size SizeRequest SizeResponse Size returns the size of the log
    Exists ExistsRequest ExistsResponse Exists checks whether an index exists in the log
    Append AppendRequest AppendResponse Appends appends an entry into the log
    Get GetRequest GetResponse Get gets the entry for an index
    FirstEntry FirstEntryRequest FirstEntryResponse FirstEntry gets the first entry in the log
    LastEntry LastEntryRequest LastEntryResponse LastEntry gets the last entry in the log
    PrevEntry PrevEntryRequest PrevEntryResponse PrevEntry gets the previous entry in the log
    NextEntry NextEntryRequest NextEntryResponse NextEntry gets the next entry in the log
    Remove RemoveRequest RemoveResponse Remove removes an entry from the log
    Clear ClearRequest ClearResponse Clear removes all entries from the log
    Events EventRequest EventResponse stream Events listens for change events
    Entries EntriesRequest EntriesResponse stream Entries lists all entries in the log
